# Interaction #1

**Contexte :** en terminal, dessin de ligne entre deux points.

**Prompt :** "Write a Python function that takes as input two points (x1, y1) and (x2, y2) and returns a list of all the points (x, y) that lie on the straight line."

**Réponse (résumé):** code Python utilisant l'algorithme de Bresenham pour générer les points entre deux coordonnées données.
